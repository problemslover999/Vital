# Vital ⚡

Your health buddy and routine tracker for a better lifestyle. Designed with a vibrant, high-energy aesthetic to keep you moving.

## [🚀 Open Live App](https://problemslover999.github.io/Vital/)

---

### Key Features
- **Vital Buddy**: A supportive assistant that motivates and guides your health journey.
- **Routine Tracker**: Tick off your daily missions and build long-term streaks.
- **Progress Insights**: Visualize your consistency with easy-to-read charts.
- **Smart Motivation**: Personalized generated messages to start your day right.

### Tech Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Animations**: Framer Motion
- **AI Engine**: Google Gemini API
- **Charts**: Recharts
